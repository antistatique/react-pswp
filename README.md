## Install

```bash
yarn add react-pswp
// or
npm install --save react-pswp
```

## Contribute

First, install all dependencies:

```bash
$ yarn
$ cd ./example && yarn
```

To ease both module and example contributions, start the following commands in paralel.

```bash
$ yarn start
# AND
$ cd ./example && yarn start
```
